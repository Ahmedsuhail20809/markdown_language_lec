# Markdown language 
> ## Q:What is markdown language?
**Markdown** is a simple , lightweight **markup language** used to **format text**.

It let's you to write **plain text**  that can be converted in to **HTML** to style content like:

* Heading
* bold/italic text
* lists
* links
* code blocks
* images tables
  
>## Q: Why Use Markdown?
* Super easy to learn
* Used in, **GitHub**, **Jupyter Notebook**, **blogs**, **decumentation**, **chat apps** etc.
* keep files clean and readable

|bold| **bold** or __bold__|

| code(inline)| `x = 3+5`|
